### 1. Tree
* 핵심 키워드: 계층, 정렬, 재귀
* 트리 자료구조/탐색 알고리즘 사용 예시: git, 파일-디렉터리 구조, 계층형 질의, DB (인덱스) 등

* `union` 연산의 비용 문제: 랭크를 도입하여 해결
