### 2. 문제 풀이 요약
#### 문제 46. N-퀸
* 대표적인 백트래킹 유형 중 하나
* 행이나 열을 하나씩 늘려가며 퀸의 배치 가능한 자리를 탐색
* 대각선 처리
  * 저자 방법: `diag1`, `diag2` 배열 생각
  * 다른 방법: `Math.abs` 이용 (이 방법이 더 깔끔한 듯)

#### 문제 47. 양궁 대회
