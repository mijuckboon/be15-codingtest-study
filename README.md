# be15-codingtest-study
BEYOND SW캠프 코딩 테스트 스터디를 위한 GitHub 저장소

### 디렉터리 구조
```
src
ㄴ main
  ㄴ docs : 스터디 노트
  ㄴ java : java 코드
ㄴ test : 테스트 코드
```
* 프로그래머스에 수록된 문제에 대한 풀이 코드 링크는 주로 [알고리즘 문제 풀이 저장소](https://github.com/mijuckboon/Algorithm)에서 시트에 공유
* 본 저장소는 저자가 출제한 문제에 대한 풀이 코드 등의 파일을 별도로 관리
